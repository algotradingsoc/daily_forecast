#  Gaussian Processes

This is the foler within the daily forecast team where Gaussian Processes are being researched.

For preliminary prototyping, end-of-day stock data from TSLA is being used, gathered from Yahoo finance.

### Dependencies
* GP Flow 2.1.3
* Numpy 1.18.4
* Pandas 1.0.3
* Tensorflow 2.3.1
* Matplotliob 3.2.1
